# E-Commerce Checkout Redesign Case Study

This project showcases a full end-to-end business analysis package for a large e-commerce company seeking to reduce checkout abandonment, improve mobile UX, and modernize the payment flow.

Deliverables include:
- Full BRD  
- Full FRD  
- 40+ User Stories  
- 80+ Acceptance Criteria  
- UAT package  
- KPIs  
- Risks  
- Process flows (AS-IS & TO-BE)  
- Architecture overview  
- High-level wireframes
